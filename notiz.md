# catch_ohl Kurzuebersicht

- `dict_tree_examples.py`: `inorder`, `preorder`, `postorder`, `levelorder` - traverses nested dictionary trees like `simpleBinaryTree`.
- `getters_setters.py`: `ExamNode` getters/setters, `connect_left` - shows class attributes, properties, and child/parent links.
- `json_load_dump.py`: `nil`, `new_node`, `insert`, `preorder_items`, `dump`, `load`, `inorder_keys` - saves and loads a flat JSON key/value tree.
- `rb_access_methods.py`: `RBTree.max`, `min`, `find`, `between`, `visit_post_order`, `visit_level_order` - class-based exam access methods on RB-style nodes.
- `rb_delete_examples.py`: `search`, `minimum`, `transplant`, `delete`, `fix_delete`, rotations - shows the RB delete flow and fix cases.
- `rb_insert_examples.py`: `insert`, `fix_insert`, `left_rotate`, `right_rotate`, `inorder` - shows RB insert and repair cases.
- `rb_validation.py`: `count_color`, `validate_bst_order`, `black_height`, `no_red_red`, `validate_rb_tree` - checks basic RBTree rules.
- `search_min_max_between.py`: `find`, `min_key`, `max_key`, `between`, `insert_bst` - practices search and interval logic.
- `simple_node.py`: `Node`, `nil`, `leaf` - shows RED/BLACK/NIL node basics.
- `traversals.py`: `inorder`, `preorder`, `postorder`, `levelorder`, `visit_post_order`, `visit_level_order` - practices traversal orders.
