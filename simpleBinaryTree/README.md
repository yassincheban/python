# Simple Binary Tree

**lesen sie diese Datei bis zum Ende!**

## Kontext
Sie sollen mit einfachsten Mitteln in Python einen Binären Baum aufbauen. Dazu sollen sie nur Boardmittel nutzen.

## Aufgabe:
Datei `simpleBinaryTree.py`
* Erzeugen sie eine Datenstruktur aus verschachtelten Hashes zur Abbildung eines Binärbaums in der Funktion `get_tree`
* Die Daten des Binärbaums finden sich in der Datei `tree.txt`. Bauen sie zunächst von Hand (auf Papier) den Baum 
  entsprechend er Reihenfolge in der Textdatei auf und tippen ihn dann erst in den Python Code ein.
* laufen sie in inorder, preorder, postorder und levelorder Reihenfolge über den Binärbaum welchen sie aufgebaut haben
  und erzeugen sie einen String welcher die Werte der Knoten des Baums enthält. Jeder Wert soll dabei mit einem 
  Zeilenumbruch `\n` getrennt werden.

## Hinweise zur Klausur:
* **Wechseln sie in das richtige Projekt Verzeichnis** (Drei Striche oben links drücken -> File -> Open Folder
  und dann das Projektverzeichnis auswählen)
* **Arbeiten sie sich schrittweise vor!**
* Sie müssen nichts vollständig neu schreiben. Wenn sie also auf die Idee kommen sollten viel neuen Code zu schreiben: 
  **Denken sie erst nach!**

## Hinweise zur Aufgabe:
* Wenn ihr Bildschirm groß genug ist, starten sie die Terminals im Split Modus
* Wenn sich ihre Software komisch verhält, überprüfen sie ob diese nicht noch 
  irgendwo anders läuft. Schließen sie ggf. alle Terminals!
